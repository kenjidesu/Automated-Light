# Automated-Light
An Automated Light that is controlled by remote and receive by the IR receiver sensor that is sent to a microcontroller to communicate with the servo.

## Hardware Used:
- Arduino Nano (clone)
- Servo SG92R
- IR Receiver
- Remote Control
- Wires
- Soldering Iron
- 3d printer

## Software Used:
- Cura (slicer)
- Arduino IDE (programming)
- Fusion 360 (3d modelling)
